# Market State & Risk Analytics Platform (MSRAP)

Research-grade analytics platform for studying:
- Derivatives market state and structure (India)
- Equity factor-driven risk decomposition (India)

## Scope
This project is developed strictly for academic and research purposes.
It does not provide trading signals, investment advice, or execution capabilities.

## Tech Stack
- Java 17 + Spring Boot
- Python (quant analytics)
- PostgreSQL / TimescaleDB
